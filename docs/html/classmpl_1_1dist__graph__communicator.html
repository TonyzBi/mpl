Equality types for communicator comparison.

Enumerator
identical 

communicators are identical, i.e., communicators represent the same communication context

congruent 

communicators are identical, i.e., communicators have same the members in same rank order but different context

similar 

communicators are similar, i.e., communicators have same tha members in different rank order

unequal 

communicators are unequal, i.e., communicators have different sets of members
