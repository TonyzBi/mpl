Detailed Description

template<std::size_t dim, typename T, typename A = std::allocator<T>>
class mpl::distributed_grid< dim, T, A >

Local portion of a distributed data grid including local overlap data.

Parameters
dimnumber of dimensions of the data grid, may be 1, 2, 3 or 4
Template Parameters
Tdata type that is hold at each grid point
Amemory allocator
Examples
distributed_grid.cc, distributed_grid_scatter_gather.cc, heat_equation_Jacobi_method.cc, and heat_equation_successive_over-relaxation.cc.
